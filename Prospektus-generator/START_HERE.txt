🚀 STARTA APPLIKATIONEN:

1. BACKEND (Terminal 1):
   cd prospektus-generator/backend
   npm install
   
   # Skapa .env fil med din Anthropic API key:
   echo "ANTHROPIC_API_KEY=din-api-key-här" > .env
   
   npm start
   → Körs på http://localhost:3001

2. FRONTEND (Terminal 2):
   cd prospektus-generator/frontend
   npm install
   npm run dev
   → Körs på http://localhost:3000

3. ÖPPNA I WEBBLÄSARE:
   http://localhost:3000

🐛 TROUBLESHOOTING:

Problem: Backend startar inte
Fix: Kolla att .env finns med giltig ANTHROPIC_API_KEY

Problem: Frontend hittar inte backend
Fix: Kontrollera att backend körs på port 3001

Problem: PDF genereras inte
Fix: Se backend logs för fel

Problem: AI-generering misslyckas
Fix: Verifiera API key och att du har credits
