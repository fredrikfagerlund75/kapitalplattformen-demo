# SNABBSTART - Kapitalplaneraren v2.0

## 🚀 Kom igång på 5 minuter

### 1. Installera beroenden

**Backend:**
```bash
cd backend
npm install
```

**Frontend:**
```bash
cd frontend
npm install
```

### 2. Konfigurera API-nyckel

Skapa `backend/.env`:
```
ANTHROPIC_API_KEY=sk-ant-api03-...
PORT=3001
```

### 3. Starta applikationen

**Terminal 1 (Backend):**
```bash
cd backend
npm start
```
✅ Backend körs på http://localhost:3001

**Terminal 2 (Frontend):**
```bash
cd frontend
npm start
```
✅ Frontend öppnas automatiskt på http://localhost:3000

## 🐛 Felsökning

**Backend startar inte:**
```bash
# Kontrollera att API-nyckel är korrekt
cat backend/.env

# Installera om dependencies
cd backend && rm -rf node_modules && npm install
```

**Frontend kan inte nå backend:**
- Kontrollera att backend körs på port 3001
- Kontrollera CORS-inställningar i `backend/server.js`

**PDF-genereringen misslyckas:**
- Kontrollera att PDFKit är korrekt installerat
- Verifiera att all data i formData är giltig (inga undefined-värden)
